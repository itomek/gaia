## Prerequisites

| Requirement | Supported |
|-------------|-----------|
| **NPU** | AMD Ryzen AI 300-series |
| **GPU** | AMD Radeon RX 7000/9000-series, Radeon PRO W-series |
| **CPU** | Any x86_64 processor (slower, but works without dedicated AI hardware) |
| **OS** | Windows 11, Ubuntu 24.04+ |
| **RAM** | 16GB minimum, 64GB recommended |
| **Python** | 3.10 - 3.12 (uv installs automatically) |

<Note>
  For detailed hardware specs, see the [Lemonade supported configurations](https://github.com/lemonade-sdk/lemonade?tab=readme-ov-file#supported-configurations).
</Note>

## Install GAIA

Choose your platform and installation type, then follow the steps below.

<Tabs>
  <Tab title="Windows">
    Installs [amd-gaia](https://pypi.org/project/amd-gaia/) from PyPI.

    ### Step 1: Install Lemonade Server

    Download and install from [lemonade-server.ai](https://lemonade-server.ai). GAIA starts it automatically when needed.

    ### Step 2: Install uv

    [uv](https://docs.astral.sh/uv/) is a fast Python package manager (10-100x faster than pip).

    Open **PowerShell** and run:
    ```powershell
    powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
    ```

    <Tip>
      If `uv` is not recognized after install, restart PowerShell or run:
      ```powershell
      $env:Path = "$env:USERPROFILE\.local\bin;$env:Path"
      ```
    </Tip>

    Verify uv is installed:
    ```powershell
    uv --version
    ```

    ### Step 3: Create Project Directory

    ```powershell
    mkdir my-gaia-project; cd my-gaia-project
    ```

    ### Step 4: Create Virtual Environment

    ```powershell
    uv venv .venv --python 3.12
    ```

    <Note>uv will automatically download Python 3.12 if not already installed.</Note>

    ### Step 5: Activate the Environment

    ```powershell
    .\.venv\Scripts\Activate.ps1
    ```

    <Warning>
      If you see a script execution error, run this once:
      ```powershell
      Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
      ```
      Then retry the activation command.
    </Warning>

    You should see `(.venv)` in your terminal prompt when activated.

    ### Step 6: Install GAIA

    ```powershell
    uv pip install amd-gaia
    ```

    <Tip>
      **Optional extras:** `uv pip install "amd-gaia[talk,rag]"` for voice and document Q&A features.
    </Tip>

    ### Step 7: Verify Installation

    ```powershell
    gaia -v
    ```

    <Success>
      You're ready to build your first agent! Continue to the [Quickstart](/quickstart).
    </Success>

    <Tip>
      **Having issues?** Check the [Troubleshooting](/reference/troubleshooting) guide, [create an issue](https://github.com/amd/gaia/issues) on GitHub, or contact us at gaia@amd.com.
    </Tip>
  </Tab>
